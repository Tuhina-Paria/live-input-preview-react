# 🚀 Live Input Preview (React)

## 📌 Overview
This project is a React-based Live Input Preview application that demonstrates controlled components and real-time UI updates. Users can type text and instantly see the output along with a live character count.

---

## 🔧 Tech Stack
- React (useState Hook)
- Tailwind CSS

---

## ✨ Features
- ⌨️ Real-time text preview while typing  
- 🔢 Live character count
- 🎯 Reset Button to clear input
- 🎯 Controlled input handling  
- 🎨 Clean and responsive UI  

---

## 💡 what I learn
- Understand controlled components in React  
- Managing user input using state  
- Real-time UI updates and component re-rendering  
- Building interactive UI with clean design
